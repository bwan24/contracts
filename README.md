# 远景能源合同智审系统

## 项目概述

远景能源合同智审系统是一个AI驱动的合同风险识别与管理平台，旨在帮助企业高效审查合同、识别潜在风险，并提供智能分析和建议。系统主要面向风电和储能领域的合同管理，支持合同上传、风险分析、规则管理等核心功能。

## 核心功能

### 1. 工作台
- 合同统计概览（总数、已完成、审核中）
- 风险统计分析（高风险项数量）
- 最近处理合同列表

### 2. 合同处理
- 支持风电和储能合同上传与管理
- 合同状态跟踪（待处理、审核中、已完成）
- 标准/非标准合同分类
- 合同搜索与筛选功能

### 3. 风险审核
- 多维度风险分析（商务、法务、技术风险）
- 风险等级评估（高、中、低）
- 风险点详细信息展示
- 风险报告导出功能（CSV、JSON、文本摘要）

### 4. 规则管理
- 灵活的审核规则配置
- 规则分类管理（商务条款、法务条款、技术条款）
- 规则导入/导出功能
- 规则搜索与筛选

## 技术栈

### 前端技术
- **框架**: Next.js 16 + React 19
- **样式系统**: Tailwind CSS 4 + tailwind-merge + clsx
- **UI组件库**: Radix UI + 自定义组件
- **状态管理**: React Hooks
- **表单处理**: React Hook Form + Zod
- **图表库**: Recharts
- **图标**: Lucide React
- **主题**: next-themes

### 开发工具
- TypeScript
- ESLint
- PostCSS

## 安装与运行

### 前提条件
- Node.js 18.17或更高版本
- npm、yarn或pnpm

### 安装步骤

1. 克隆项目
```bash
git clone <项目仓库URL>
cd <项目目录>
```

2. 安装依赖
```bash
# 使用npm
npm install

# 或使用pnpm
pnpm install

# 或使用yarn
yarn install
```

3. 开发环境运行
```bash
npm run dev
# 或
pnpm dev
# 或
yarn dev
```

4. 构建生产版本
```bash
npm run build
# 或
pnpm build
# 或
yarn build
```

5. 运行生产版本
```bash
npm start
# 或
pnpm start
# 或
yarn start
```

## 系统使用指南

### 合同管理流程
1. 在"合同处理"页面上传合同文件
2. 系统自动进行风险分析（模拟）
3. 在"风险审核"页面查看详细的风险分析结果
4. 根据系统建议进行合同修改
5. 导出风险报告存档

### 规则管理
1. 在"规则管理"页面查看现有审核规则
2. 可以添加新规则、编辑或删除现有规则
3. 支持规则的导入导出，便于规则集的备份和迁移

## 系统特点

1. **AI驱动**: 基于预设规则智能识别合同风险点
2. **多维度分析**: 从商务、法务、技术等多角度评估合同
3. **灵活配置**: 支持自定义审核规则和风险等级
4. **用户友好**: 直观的界面设计和流畅的用户体验
5. **导出功能**: 支持多种格式的报告导出
6. **响应式设计**: 适配不同屏幕尺寸的设备

## 注意事项

1. 当前版本使用模拟数据，实际部署时需要对接后端API
2. 系统支持的合同类型主要为风电和储能领域的标准/非标准合同
3. 风险分析结果仅供参考，最终决策需专业人员确认
